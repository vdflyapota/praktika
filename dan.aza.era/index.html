Why do you need a strong password?

Password security is of great importance right now. It is estimated that more than 800,000 people are hacked every year. Technologies such as 2FA and FaceID provide greater security on the Internet, but if the service does not support such security features, a good strong password is a proven way to protect your online accounts.

Another good practice is not to use your passwords on any website without an HTTPS certificate. But even if the website looks good and has certificates, it's useful to be careful and not use your legitimate credentials for any website that you're not absolutely sure about. And a good strong random password ensures that even if you compromised your password on a bad website, all your other accounts will not be exposed to hackers.

What about our password verification, is it safe? We do not store your password, all verification is performed on the online page you are on, and not on the server. We are interested in making users more secure on the Internet, so we make sure that every verified password is not transmitted over the Internet.

Password Security Checker

A Password Security Checker is a tool or service that assesses the strength and security level of a password. By using a Password Security Checker, users can determine if their passwords are strong enough and take necessary steps to improve their online security.

Password Requirements:

  • The password must be long. Preferably at least 8 characters. Better - from 16.
  • The password must be unique for each account. Do not use the same password multiple times.
  • Use lowercase and uppercase letters.
  • Use numbers.
  • Use symbols.
  • Don't use dictionary words. For example: "cyber", "book", "monitor". It is better to have such a plan: "yvrvne", "gfazhkshr" and so on.
  • Don't use empty passwords.
  • Use (if possible) a combination of several alphabets. For example, Russian, Kazakh, English.
  • Change particularly important passwords periodically. For example, once a month.
  • When creating a new password, make sure that it differs from all the others. Moreover, this difference should be significant.
  • Preferably a password that does not have a logical sequence of characters. For example, "qwerty", "asdfg", "12345" and the like.
  • Avoid using personal information such as your name, username, birthdate, etc., in your password.

Atbash Cipher

The Atbash Cipher simply reverses the plaintext alphabet to create the ciphertext alphabet. That is, the first letter of the alphabet is encrypted to the last letter of the alphabet, the second letter to the penultimate letter and so forth.
